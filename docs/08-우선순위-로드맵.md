# 08. 우선순위 & 로드맵

## P0 — 출시 필수

- **계정·팀**: 카카오 로그인, 팀 생성·초대(코드 만료 설정), 마이페이지
- **회의 전**: 회의 생성, 아젠다 자동 생성, 알림
- **회의 중**: 회의 룸(다인 접속·미접속 경고), Moonshine 로컬 STT 통합(VAD 분할 포함), **안건 추적 ★**, **실시간 기여도 바 ★**, 결정·액션 빠른 입력
- **회의 후**: 그루핑 정리, AI 보조 추출, 검토·확정, 기여도 대시보드(3축 지표+신뢰도), 액션 대시보드
- **알림**: 인앱 + 브라우저 푸시 + 이메일, 회의 5분 전
- **운영**: 랜딩 페이지, 약관

## P1 — 출시 후

- 사전 의견 수집 폼
- 회의 2시간 전 준비 체크 푸시, 준비도 현황판
- 마감일 자동 푸시
- 교수 제출용 PDF
- 기여도 리포트 내보내기

## V2

- AI 자동 안건 추적
- 종합 점수 표준화 (팀 내 정규화)
- 액션 난이도 가중치
- 동료 평가
- Discord/Zoom 봇 직접 연동
- 모바일 대응
- 자동 업데이트 인프라
- 오프라인 대면 회의 지원 (현재는 온라인 회의 보조에 집중)
- 과제 업로드·검토 (클라우드 드라이브 연동)
- AI 챗봇 (회의 상황·일정 확인)
