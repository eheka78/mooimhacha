# 06. 기여도 산정 로직

회의에서 기여도는 **3축(태스크 완료 / 발언 비중 / 참석)** 으로 측정한다.

## 설계 원칙

1. **3축 분리 측정 + 종합 표시** — 세 축을 독립 측정하고, 종합 점수는 세 축의 가중 합으로 산출
2. **비율 기반** — 회의 길이·팀 규모와 무관하게 비교 가능
3. **기회 보정** — 측정 불가 축은 0점 처리 금지, 가중치 재분배
4. **명시적 보정** — 모든 보정에 라벨 표시
5. **팀별 가중치 조정** — 기본 가중치는 팀 설정(TeamSettings)에서 조정 가능

## 3축 지표

| 축 | 기본 가중치 | 구성 요소 | 측정 시점 |
| --- | --- | --- | --- |
| 태스크 완료 | 50% | 액션 아이템 완료율 + 마감 준수율 | 회의 후 |
| 발언 비중 | 30% | 발언 비중 (참여 시간 보정) | 회의 중·후 |
| 참석 | 10% | 출석 시간 비율 + 정시 도착(지각 반영) | 회의 중 |

- 기본 가중치 합계는 90% — 종합 점수가 **적용된 가중치로 정규화**되므로(아래 공식) 합이 100%가 아니어도 무방하다.
- 가중치 비율은 팀마다 조정 가능 (TeamSettings).

## 축별 산정

### 태스크 완료 축 (50%)

회의에서 배정받은 액션 아이템의 완료 여부와 마감 준수를 합산한다.

액션 완료율:
```
완료율 = 완료한 액션 / 배정받은 액션
(배정 액션 0개 → 측정 불가, 가중치 재분배)
```

마감 준수율:
```
액션별 점수:  마감 전 100 / 당일 80 / 1일 초과 60 / 3일 초과 30 / 7일+ 0 / 미완료 0
마감 준수율 = 액션별 점수 평균
```

태스크 완료 점수 = 액션 완료율과 마감 준수율의 가중 평균 (내부 배분 비율은 미확정 — [09](09-미결정-사항.md)).

### 발언 비중 축 (30%)

```
조정된 발언 비중 = 본인 글자수 / (본인 참여 시간 동안의 전체 참여자 글자수)
```
→ 늦게 들어왔거나 일찍 나간 사람이 불리하지 않도록, 본인이 있던 시간대의 글자수만 분모로 사용.

### 참석 축 (10%)

출석 시간 비율:
```
실제 참여 시간 = (마지막 leave - 첫 join) - 자리비움 누적
출석 비율     = 실제 참여 시간 / 회의 총 시간
```

정시 도착 (지각 반영):
```
허용 지각 분 = 회의 길이 × punctuality_grace_ratio (기본 10%)
정시 점수    = max(0, 1 - 지각 분 / 허용 지각 분)
```
- 지각이 누적될수록 참석 축 점수가 비례 감소한다 (자동 감점).

## 결석·저참여 처리

- **최소 참여 기준**: 회의 참여 시간이 회의 총 시간의 **40% 미만**이면 정상 참여로 보지 않고, 해당 회의 기여도를 자동 하향 조정한다.
- **장기 결석**: 무단 결석 회의는 해당 회의 기여도 0점으로 누적에 포함. 사유 결석은 팀 설정에 따라 제외 옵션.

## 종합 점수

```
종합 = Σ(축 점수 × 가중치) / Σ(적용된 가중치)
```
측정 불가 축은 분모에서 제외 → 남은 가중치가 비례 확장(재분배)됨.

## 신뢰도 라벨

| 등급 | 조건 |
| --- | --- |
| High | 3축 모두 측정 + 회의 30분 이상 + 오디오 캡처 손실 5% 미만 |
| Medium | 2축 측정 또는 회의 15~30분 또는 일부 캡처 손실 |
| Low | 1축 이하 측정 또는 회의 15분 미만 또는 캡처 손실·이상 이벤트 다수 |

## 누적 점수

```
누적 점수 = Σ(회의 점수 × 회의 시간) / Σ(회의 시간)   # 회의 시간 가중 평균
```
- 사유 결석: 팀 설정에 따라 제외 옵션
- 무단 결석: 0점으로 포함

## 처리 정책 우선순위 (점수 산정 시 순서)

1. 데이터 무결성 검증
2. 신원·권한 검증
3. 시간 동기화
4. 사유·예외 처리
5. 측정 가능성 판단
6. 상한·보정 적용
7. 재분배 계산
8. 신뢰도 라벨링
